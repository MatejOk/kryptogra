Wysłanie aktualizacji (Magiczna trójka komend)
Kiedy wprowadzisz zmiany i zapiszesz pliki (Ctrl + S), klikasz w dolne czarne okienko (terminal) w Visual Studio Code. Jeśli serwer działa, zatrzymaj go (Ctrl + C i Y).

Następnie wpisujesz po kolei te trzy komendy (po każdej wciskając Enter):

git add .
(To polecenie "pakuje" wszystkie zmienione przez Ciebie pliki).

git commit -m "Dodanie nowych zagadek o wezlach"
(To polecenie nadaje paczce nazwę/etykietę. W cudzysłowie wpisujesz, co krótko zrobiłeś, żeby w przyszłości wiedzieć, co to była za aktualizacja).

git push
(To polecenie "wysyła" zapakowane zmiany na serwery GitHuba).
